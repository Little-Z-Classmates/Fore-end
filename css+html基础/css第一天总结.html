1. 行内式 , 内嵌 , 外链


2. font-size:字体的大小( px:像素,默认是16px,最小是12px; em:相当于当前默认字体的倍数; % ) ; font-color:颜色 ;
    font-family:"字体样式"(ps:多字体方案:"XX,XX2,XX3," 如果字体库xx没有,就用xx2)
    font-weight:bold(加粗)/normal(默认)/数字(400正常)
    font-style:italic(倾斜)/normal(默认)
    (ps:font连写: 字体加粗和倾斜 可以调换位置和省略; px和字体样式 不能调换位置和省略)


3. text-align:left/right/center/bottom(ps:水平对齐方式)


4. line-height:行高 (ps:垂直效果，如果没设置height,就设置行高，它的高就是行高.)
    行高等于元素高，文字垂直居中
    行高大于元素高，文字垂直偏下
    行高小于元素高，文字垂直偏上


5. text-indent:2em ; (ps:文字首行缩进)


6. text-decoration:none/underline/overline/line-through:删除线


7. div.class+enter:快速生成
    div.class+enter:快速生成
    div.id+enter:快速生成
    .id+enter:快速生成
    #id+enter:快速生成


8. 后代选择器，后代全部变；
    子代选择器，只有子代才有，子代的后代没有，ps：如果子代和子代的子代的标签一样也是可以继承的
