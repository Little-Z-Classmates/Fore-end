1. “ | ” 可以用li 标签 中的 width：1px； height：16px； 再加个 background-color:red;


2. 静态定位是标准流没办法边偏移属性比如top：20px / fixed ：固定定位 / relative:相对定位 / absolute ： 绝对定位


3. 布局三等级（从低到高）：标准流 < 浮动流 < 定位流


4. fixed ： 固定定位 。 完全脱离标准流 、 可以实现模式转换--行内块 、 不随页面滚动而滚动 、 可以使用margin
     ( 固定定位的参照物 ： 浏览器的可视区域 )

5. absolute : 绝对定位 。 完全脱离标准流 、 模式转换--行内块
     ( 绝对定位的参照物 ：有定位 的父盒子 / 如果没有这样的父盒子，继续往外找一层，最近优先 ，最后找到body ) (一般不会单独出现，经常配合父盒子一起出现 ： 子绝父固 、 子绝父绝 、 子绝父相)


6. relative : 相对定位 。元素相对于它原来在标准流中的位置,有了边偏移量相对于标准流中的位置脱标(原有区域仍会保留) 、不会转换模式 、 正常情况下不会让它做位移 、
      ( 大多数： 子绝父相 )


7. 定位的层级关系 : 默认情况下定位有层级， 结构代码越往后，层级越高
     ( z-index : 5 这个数值越高 层级越高) 也可以利用相对定位
