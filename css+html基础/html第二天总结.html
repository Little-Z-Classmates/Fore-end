4. 表格： table , th:表头 放在tr里面 , tr:表格的行 , td：行内的单元格
    结构： table ， caption: 表格的标题 里面可以放h1 ，thead：th表头放里面 ，tbody: td放里面
    属性： width,height: 宽高.    border: 表格的边框.   cellspacing: 边框之间的空隙,  
cellpadding: 内容与边框距离,一般不用.   align:控制元素的水平对齐方式(left,right,center)
rowspan: 跨行合并，在td中，后面不需要数据就删除，不然后面数据会被挤出去
colspan: 跨列合并，在td中，后面不需要数据就删除，不然后面数据会被挤出去


5. 表单： < from > </from >
文本: <input type="text" value="默认值(记住账号密码)实体字" placeholder="占位符，输入字就不见，虚体字">
密码: <input type="password" maxlength="6" >(ps:最大maxlength最大字符数)
单选: <input type="radio" name="xxx" >男 <input type="radio" name="xxx" >女 <input type="radio" name="xxx" checked>保密
(ps:由于单选互斥，所以需要一个以相同name分为一组,checked 是默认选中的)
多选: <input type="checkbox" name="xxx" >扫雷 <input type="checkbox" name="xxx" >坦克大战 <input type="checkbox" name="xxx" checked >蜘蛛纸牌
(ps:checked 是默认选中的)
普通按钮: <input type="button" value="按钮名字" >
提交按钮: <input type="submit" (默认名字是"提交") >
重置按钮: <input type="reset" value="按钮名字" >(ps:必须和from表单使用)
图片按钮: <input type="image" src="图片地址" >
新版本按钮: <button > 按钮名字 </button >
文件域: <input type="file" >(ps:用于提交文件)
下拉列表: <select> <option selected> xxx </option> </select> (ps:selected是默认)
文本域: <textarea cols="100" rows="100"> </textarea >
