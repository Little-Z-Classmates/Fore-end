1. 结构:html 表现:css 行为:js


2. 标题:h1-6 (ps:标题一般只有一个h1,一般作为网页logo)


3. p标签有个空格合并现象,空格多就只显示一个,如果需要打空格可以(pre)或者(& nbsp;)


4. 水平线用 hr 可以设置宽度, 重点!!:这是一个寂寞的天,下着有些伤心的雨


5. 布局标签: div是一个块级元素 span是一个行内元素


6. 格式化标签: 重点背下来 (b---strong 加粗) (u---ins 下划线) (i---em 斜体) (s---del 删除线)


7. 图片: <img src="地址" alt="无图片显示" title="鼠标放图片中显示的文本" width或者height="..px">
    (ps: width和height只写一个那么另一边等比例缩放.)


8. 链接: <a href="url" target="怎么显示">链接文本或者图片<a>
    (ps: target:_self:自己界面,_blank:新界面 , base在head中设置:<base href="基础地址" target="_xxx"> )


9. 链接外网地址请保证 域名 完整,href="#"是空链接,但是如果 a 中有 ID 值 ,#id跳转到相关的url


10. 相对路径: ① 当前文档下: src="xxx.jpg"
    ② 上一个文件夹: src="../xxx.jpg"
    ③ 下一个文件夹: src="img/xxx.jpg"
    绝对路径: src="xx盘下的一大串文件名下.jpg"
