1. 交集选择器:找到一个元素,拥有所有的条件;既有标签一又有标签二
    比如：p.one 选择的是：类名为 .one 的段落标签。


2. 并集选择器:满足任何一个条件都可以被使用 , 当很多元素都拥有一个元素 写到并集选择器里面.


3. 伪类选择器:(:link 没有被访问的时候 / :vistied 被访问之后 / :hover 悬停的时候 / :active 点击不松手的时候)


4. ul>li*8{这是一个寂寞的天$} (ps:快捷键,ul里面的li 8次)
    li:first-child{xxxx;}找到第一个元素/
    li:last-chlid{xxx;}找到第二个元素/
    li:nth-chlid(2){xxx;}找到第某个元素/
    li:last-chlid(odd){xxx;}找到奇数个元素/
    li:last-chlid(even){xxx;}找到偶数个元素/
    li:last-chlid(3n){xxx;}找到某个倍数个元素/


5. 块级:独占一行,可以设置宽高,没设置宽度的话,和父盒子一样
    行内:一行可以放多个行内元素,不可以直接设置宽高,如果回车,会有一个空格
    行内块:可以设置宽度和高度,默认宽度为:它本身内容的宽度


6. 背景:backgrond-image是背景,不是内容,显示的话需要宽高;
    img src="" alt="" 是内容,自带宽高
    background-repeat:no-repeat/repeat-x/repeat-y 默认平铺
    background-size:xx% xx% ;背景的百分比
    background-positon:9宫格 = 水平:left center right / 垂直: top center bottom
    background-positon:( x,y ) 50px 30px ;
    background-attachment:fixed固定/scroll默认滚动
    background:连写 red url("") no-repeat center center 顺序没有要求
    ( ps:如果不经常换的用background, 经常换的用img )
